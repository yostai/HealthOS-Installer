# HealthOS Installer

Your personal AI health coach — running 24/7 in the cloud, checking in via Telegram.

---

## Before You Start

You'll need:
- **Claude desktop app** (free — get it at claude.ai)
- **A credit card** for AWS — your first 3 months of server time are **FREE**
- **Telegram** on your phone or desktop (free — App Store or Google Play)
- **An Anthropic API key** (free to start — get one at console.anthropic.com)

---

## Setup — 3 Steps

**Step 1: Install Claude desktop app**
Go to **claude.ai** and download the desktop app for Mac.

**Step 2: Open this folder in Claude**
Open Claude desktop → click the **Code** tab at the top → click **"Set workspace folder"** → select the folder you just downloaded and unzipped.

**Step 3: Type /install and press Enter**
Claude will guide you through the rest. Total time: about 20 minutes.

---

## What You'll Get

- A Telegram bot that checks in with you about your health daily
- Your data stored securely in your own private cloud
- Monthly cost: **~$7** (free for the first 3 months)
